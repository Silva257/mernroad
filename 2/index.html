Month 1 — HTML Fundamentals (Master & Learn)

Master: Semantic HTML, document structure (<header>, <main>, <nav>, <section>, <footer>), forms basics, accessibility basics (aria, alt, headings).

Learn: HTML5 APIs overview (localStorage, history API basics), meta tags, favicons.

Aware: Web Components (basic idea).

Project: Simple personal portfolio HTML skeleton + contact form (no backend yet).

Milestone: Comfortable writing accessible, semantic HTML pages and structuring content.

Resources: MDN HTML guide, freeCodeCamp Responsive Web Design (HTML).

Month 2 — CSS Fundamentals (Master)

Master: Box model, selectors, specificity, layout basics (display, position, float), typography (font stacks, rem/em), colors, responsive media queries.

Learn: Flexbox basics & common patterns, CSS variables (custom properties).

Aware: Preprocessors (Sass) — know when to use.

Project: Style portfolio home page; create responsive nav and hero using Flexbox.

Milestone: Produce responsive, well-structured layouts using CSS and Flexbox.

Resources: MDN CSS, CSS-Tricks guides, Flexbox Froggy.

Month 3 — CSS Grid & Advanced Layout (Master)

Master: CSS Grid (tracks, areas, responsive grids), combining Grid + Flexbox for page layouts.

Learn: Modern responsive patterns, container queries (if available), utility classes.

Aware: CSS-in-JS tradeoffs (styled-components, emotion).

Project: Build a multi-section portfolio layout (Grid-based project gallery).

Milestone: Confident in Grid for major layouts and combining with Flexbox.

Resources: CSS Grid by MDN, Grid Garden, CSS-Tricks Grid guide.

Month 4 — CSS Animations & Interactions (Master)

Master: CSS transitions, keyframe animations, using transform & opacity for performant animations, prefers-reduced-motion.

Learn: Micro-interactions (hover/focus effects, animated buttons, animated modals), CSS variables for animation control.

Aware: When to use JS animations vs CSS animations.

Project: Add polished animations to portfolio (nav transitions, hero entrance, gallery hover effects).

Resources: MDN animation docs, CSS-Tricks animation guides.

Month 5 — JavaScript Fundamentals I (Master)

Master: Language basics: variables (let/const), types, operators, control flow, functions (declaration, expression, arrow functions), scope & closures basics.

Learn: DOM basics: querySelector, event listeners, simple DOM updates, forms handling.

Aware: Older patterns (var, IIFEs) for historical context.

Project: Add interactive features to portfolio: show/hide sections, form validation (client-side), smooth scrolling.

Milestone: Build interactive pages with vanilla JS and manipulate DOM confidently.

Resources: MDN JS guide, JavaScript.info (Basics).

Month 6 — JavaScript Fundamentals II (Master)

Master: Objects, arrays & array methods (map, filter, reduce), asynchronous JS basics (callbacks, Promises), fetch API.

Learn: Error handling (try/catch), JSON handling, localStorage usage.

Aware: Generator functions & iterators (know they exist).

Project: Build a small SPA-like TODO app (CRUD in localStorage) with vanilla JS.

Milestone: Comfortable with array operations and basic async patterns.

Resources: JavaScript.info (async/tutorial), MDN Promises.

Month 7 — Version Control & Git (Master)

Master: Local Git workflow: init, add, commit, branch, merge, rebase basics, stash.

Learn: GitHub: push, pull, fork, clone, pull requests, resolving merge conflicts, basic PR review flow.

Aware: Git hooks & CI integration (know they exist).

Project: Put your portfolio into a repo; create branches for features; open a PR and merge.

Milestone: Use Git and GitHub professionally: branching, PRs, resolving conflicts.
