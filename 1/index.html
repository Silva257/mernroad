Overview — How to use this roadmap

Each month records three things: topics to Master (practice until comfortable), topics to Learn (understand & apply), and topics to Be aware (familiar, keep for later). Spend 3–4 hours daily: ~1 hour learning, 1.5–2 hours coding, 30–60min notes/tests.

All links are free. Mark each month as complete using the checkbox inside the month box; progress auto-saves in your browser.

Project: Build a marketplace or social platform with MERN backend and a React frontend + React Native mobile app that share the same API.

Core features: user auth (roles), CRUD for posts/products, search, file uploads, payments (optional), real-time notifications (WebSockets), admin dashboard, responsive frontend, mobile app with key flows (browse, create, profile, notifications).

Deployment: Frontend on Vercel/Netlify, Backend on Render or a VPS, Database on MongoDB Atlas. Add CI/CD with GitHub Actions.
